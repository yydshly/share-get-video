# V0.6.0 三路线 Baseline 对比面板

## 背景

V0.5.8 完成了 Pillow 路线完整音频链路验证。
V0.5.9 完成了 AI素材路线 baseline 体检。
Remotion 路线此前已完成完整链路验证。

三条路线的结论目前存在于各自的 docs 报告中，用户无法在 UI 上直接看到对比。

本轮目标是将三条路线 baseline 结果产品化为一个 UI 对比面板。

## 三路线 baseline 结论

### Pillow（local_frame_compose）

- **状态**：完整链路已验证
- **代表实验**：`web_local_frame_compose_fca5e1f8`
- **视频时长**：56.8s
- **优势**：生成最快、稳定性最高、可读性最好、成本最低、文字 100% 可控
- **问题**：成片感弱、Ken Burns 几不可感知、总结页停留偏短、视觉单调
- **适合**：每日资讯、批量快讯、技术摘要
- **推荐级别**：批量默认路线

### Remotion（template_programmatic_render）

- **状态**：完整链路已验证
- **代表实验**：`web_template_programmatic_render_fb270ca2`
- **视频时长**：85.3s
- **优势**：成片感最强、卡片入场动画明显、数字强调醒目、封面感强、动效丰富
- **问题**：生成慢、构建链路更复杂、KP2 时长偏长、节奏需继续优化
- **适合**：精品单片、重要选题、需要视频感的内容
- **推荐级别**：精品默认路线

### AI素材（ai_asset_then_compose）

- **状态**：完整链路已验证
- **代表实验**：`ai_asset_full_95e8de24`
- **视频时长**：72.076s
- **优势**：视觉氛围最强、科技感突出、AI背景增强电影感、信息卡仍保持可读
- **问题**：调用 MiniMax 图像 API、成本高于 Pillow/Remotion、背景一致性需要控制、非科技内容适配性未知
- **适合**：重大发布、深度分析、需要视觉氛围的视频
- **推荐级别**：视觉增强可选路线

## 当前推荐策略

```
当前不应该选唯一默认路线，而应该按场景分配路线。

1. 批量资讯：优先 Pillow
   → 稳定快速，适合每日资讯、批量快讯、技术摘要

2. 精品视频：优先 Remotion
   → 动效丰富，成片感强，适合重要选题、需要视频感的内容

3. 重要选题 / 氛围增强：可选 AI素材
   → 视觉氛围最强，科技感突出，适合深度分析、重大发布
```

## 后续路线规划

1. **Remotion 多表现范式**：探索不同 Remotion 模板风格，丰富精品路线选择
2. **AI背景 + Remotion 信息卡混合模式**：AI素材做背景，Remotion 渲染信息卡+动效，兼顾视觉氛围和信息可控性
3. **成本评估**：量化单次视频的图像生成成本，建立成本预算模型
4. **风格模板化**：固化 2-3 个 AI 背景风格模板，降低风格随机性

## 当前限制

1. 页面使用静态 baseline 数据，不读取实时 runtime
2. 报告链接为静态文件路径，前端无法直接预览 doc 内容
3. 实验ID和视频时长基于已跑过的真实实验，非自动拉取
4. 评分为主观判断，基于当前验证结果，后续可能调整
